# term-relay-sdk

Open-source SDK and extension examples for Term Relay.

## Included

- Python SDK: `extensions/python_sdk`
- Example extensions (spawn/tmux/simple-io): `extensions/*`
- Protocol and adapter docs:
  - `docs/control_protocol.md`
  - `docs/control_protocol_tmux_adapter.md`
  - `docs/session_adapter_architecture.md`

For iTerm2-specific integration docs and tools, see `term-relay-iterm2`.
